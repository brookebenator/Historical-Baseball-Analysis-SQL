# Historical-Baseball-Analysis---SQL

***Overview:***
---------------------
This database was provided through the course SQL For Data Analysis: Advanced SQL Querying Techniques by Maven Analytics. Throughout the 8.5 hours of learning advanced SQL, I build upon my previous knowledge and gained new SQL skills including:

-Window Functions  
-CTEs (Common Table Expressions)  
-Table Joins  
-Subqueries

The Lahman Baseball Database includes information about major league baseball players dating back to 1871. Utilizing these techniques, I was able to query the database and discover insights about historical baseball players, including team and salary data.
